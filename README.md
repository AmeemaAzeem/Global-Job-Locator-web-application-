#  Global Job Finder

A full-stack job search web application that allows users to search and explore job opportunities worldwide.

---

##  Features

-  Keyword-based job search (e.g. "React Developer")
-  Country-based filtering (e.g. USA, Pakistan, Remote)
-  Smart fallback system (shows alternative jobs if no exact match found)
-  Fast API integration using external job data
-  Direct "Apply" links to job postings
-  Responsive UI

---

## Tech Stack

### Frontend:
- React.js
- Axios
- CSS

### Backend:
- Node.js
- Express.js
- External Job API (Remotive API)
